# Supply-Chain-Delivery-Analysis
Supply Chain Analysis using Excel, MySQL &amp; Power BI - 180K+orders

## Project Overview
Supply chain logistics analysis of 65,752 e-commerce orders to identify delivery bottlenecks and shipping inefficiencies across global regions.
Dataset: DataCo Global — 180K+ orders across multiple regions, shipping modes and product categories.

## Tools Used
- Excel — Data Cleaning & Preparation
- MySQL — Data Analysis & Querying  
- Power BI — Interactive Dashboard

## Dashboard Features
- KPI Cards: Total Orders, Total Sales, Late Delivery Rate, Avg Delivery Days
- Shipping Mode Slicer Filter
- Order Region Slicer Filter
- Delivery Status Breakdown (Donut Chart)
- Sales by Region (Bar Chart)
- Avg Delivery Days by Shipping Mode
- Monthly Order Trend (Line Chart)

## Key Findings
- 65,752 total orders analyzed
- $12.77M total sales tracked
- 54.82% orders had late delivery
- Standard Class — slowest shipping mode
- Same Day — fastest delivery mode
- Western Europe — highest revenue region
- Avg delivery time: 3.50 days

## Project Goals
1. **Delivery Performance:** Identify late delivery patterns across shipping modes and regions.

2. **Shipping Mode Analysis:** Compare First Class, Second Class, Standard Class and Same Day performance.

3. **Regional Sales Insights:** Find highest revenue regions globally.

4. **Bottleneck Identification:** Pinpoint categories with highest delay rates.

5. **Data Visualization:** Build interactive Power BI dashboard for actionable insights.
